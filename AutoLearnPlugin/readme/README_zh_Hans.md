# 自主学习插件

LangBot 自主学习插件，从真实对话中持续学习并自适应调整机器人行为。

## 功能

- **自主学习对话风格** — 从真实对话中学习用户的表达方式、常用语和句长习惯
- **理解群黑话** — 自动统计群聊高频词，支持 LLM 推断含义
- **社交关系 + 好感度系统** — 记录互动关系、好感度（0–100）、心理状态
- **人格演化** — 随时间推移自适应调整温暖、幽默、正式、共情、好奇等人格特质
- **知识图谱 + 记忆图** — 构建记忆网络，WebUI 可视化展示

## 配置

| 配置项 | 说明 |
|--------|------|
| LLM 模型 | 可选，用于 `!learn infer` 黑话含义推断 |
| 注入学习上下文 | 是否将学习到的信息注入系统提示词（默认开启） |

## 命令

```
!learn              — 学习概览
!learn slang        — 群黑话 TOP 榜
!learn relation     — 当前用户关系档案
!learn infer        — LLM 推断黑话含义
!learn graph        — 记忆图谱统计
```

## 开发调试

```bash
cd AutoLearnPlugin
cp .env.example .env
python -m langbot_plugin.cli.__init__ run
```

在 LangBot 流水线配置中启用本插件即可。
